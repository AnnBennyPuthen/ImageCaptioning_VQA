Multimodal AI: Image Captioning & Visual Question Answering

This project implements two important vision–language tasks:

Image Captioning – generating natural language descriptions for images.

Visual Question Answering (VQA) – answering questions based on the visual content of an image.

The system uses pretrained multimodal transformer models to combine computer vision and natural language understanding, enabling machines to interpret images and respond in human language.

The implementation is built using Python, PyTorch, and Hugging Face Transformers and was developed in Google Colab.
